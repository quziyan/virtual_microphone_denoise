✅ 安装完成 — 怎么用

  1. 打开应用程序 → VibeCodingVirMic,菜单栏会出现 ● VibeCodingVirMic 图标。
  2. 点它 → Start,首次会弹麦克风权限,点允许。
  3. 在系统设置 → 声音 → 输入里选 BlackHole 2ch(或在会议/输入法里把麦克风选成 BlackHole 2ch)。
  4. 正常说话即可——背景人声会被实时去除。强度可在菜单里切 Off / Gentle / Aggressive。

不用时把"输入"切回内置麦克风,或在菜单里 Stop / 退出。VibeCodingVirMic 必须保持运行,BlackHole 才有声音。
